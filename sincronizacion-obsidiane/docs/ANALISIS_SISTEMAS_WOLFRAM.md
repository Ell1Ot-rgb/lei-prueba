## 1. Wolfram Physics Project

### 1.1 Hypergraph Rewriting (Reescritura de Hipergrafos)

El universo se representa como un **hipergrafo** - una red donde los nodos y conexiones (hiperaristas) constituyen los elementos fundamentales discretos del espacio.

```
Estado Inicial:       Regla de Reescritura:       Estado Siguiente:
    A ─── B            {{x,y},{y,z}} →           A ─── B ─── C
     \   /              {{x,z},{z,w}}                 \ |
      \ /                                              \|
       C                                                D
```

**Conceptos Clave:**
- El espacio no es continuo - es una red discreta
- El tiempo ES la aplicación sucesiva de reglas
- La materia = patrones localizados en el hipergrafo
- Las leyes físicas EMERGEN del comportamiento de las reglas

### 1.2 Aplicabilidad a Nuestro Sistema

| Concepto Wolfram | Equivalente en Sistema Perceptual |
|:-----------------|:----------------------------------|
| Hipergrafo | Red de dendritas + conexiones S4 |
| Regla de reescritura | Transformaciones en el predictor jerárquico |
| Evolución temporal | Cada `tick()` del oscilador gamma |
| Patrones localizados | Coaliciones en Global Workspace |

**Propuesta:** Representar las percepciones como nodos de un hipergrafo donde:
- Nodos = estados perceptuales
- Aristas = relaciones causales entre percepciones
- Reglas = cómo una percepción transforma otra

## 5. Multiway Systems y Branchial Space

### 5.1 Sistemas Multiway

Un **sistema multiway** permite que múltiples estados coexistan en cada paso temporal:

```
Uniway (tradicional):         Multiway (Wolfram):
     A                              A
     │                            ╱ │ ╲
     ▼                           B  C  D
     B                          ╱│╲     │
     │                         E F G    H
     ▼                         
     C                         
```

### 5.2 Branchial Space

El **espacio branchial** es el espacio definido por las relaciones entre las diferentes ramas del sistema multiway:
- Es el "espacio de los estados cuánticos"
- La **mecánica cuántica** es relatividad general en el espacio branchial
- El **entanglement** = ancestro común en el grafo multiway

### 5.3 Aplicabilidad

Nuestro sistema actualmente es **uniway** (un estado a la vez). Propuesta de extensión:

```python
# Actual (uniway):
estado = percibir(datos)

# Propuesto (multiway):
estados_posibles = percibir_multiway(datos)  # Lista de estados
estado_colapsado = workspace.votar(estados_posibles)
```

Esto permitiría:
- Mantener múltiples hipótesis perceptuales simultáneas
- "Colapsar" solo cuando hay evidencia suficiente
- Modelar la incertidumbre cuántica de la percepción

## 8. Causal Graphs y Event Horizons

### 8.1 Grafos Causales

En Wolfram Physics, el **causal graph** mapea qué eventos pueden afectar a otros:

```
    Evento A        Evento B
         \          /
          \        /
           ↘     ↙
           Evento C
              │
              ↓
           Evento D
```

### 8.2 Event Horizons como Desconexiones

Un **event horizon** = desconexión en el grafo causal:

```
Dentro del horizonte ─────×───── Fuera del horizonte
(no hay causalidad posible)
```

### 8.3 Aplicabilidad a Percepción

En nuestro sistema:
- **Horizonte perceptual** = límite de lo que puede afectar la percepción actual
- **Olvido** = desconexión causal del historial antiguo
- **Fusión multimodal** = unificación de ramas causales
