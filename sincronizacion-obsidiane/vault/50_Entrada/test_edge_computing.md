# Computación de Borde (Edge Computing) y el Sistema Sensorial

La computación de borde traslada el procesamiento de datos cerca de la fuente de generación, actuando como el sistema nervioso periférico del Organismo Vivo.

## Funciones
- **Respuesta Refleja**: Permite reacciones inmediatas a estímulos críticos sin esperar la latencia de la nube.
- **Filtrado Sensorial**: Descarta ruido (datos irrelevantes) antes de enviarlos a la memoria central, previniendo la sobrecarga.
- **Autonomía Distribuida**: Nodos locales que operan independientemente si se corta la conexión central.
