# Algoritmos Evolutivos y Adaptación del Grafo

Para que el sistema sea verdaderamente "vivo", su estructura debe evolucionar. Los algoritmos evolutivos permiten que las conexiones más útiles se fortalezcan mientras que las obsoletas se degradan.

## Proceso Evolutivo
1. **Mutación de Enlaces**: Creación aleatoria de nuevas relaciones para explorar hipótesis.
2. **Selección por Relevancia**: Los nodos con mayor centralidad de autovector tienen más probabilidades de "sobrevivir" a una limpieza de memoria.
3. **Crossover Cognitivo**: La combinación de dos subgrafos para formar una nueva estructura de conocimiento más eficiente.
